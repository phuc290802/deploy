# PRINT
- Write front-end app to manage todos.

## Tech requirements
- Use NexJS with typescript, client side redering.
- Calling API using Axios
- Use Material UI and TailWind for styling

## Features
```
# List all todos
# Create a todo
# Mark a todo as "Done"
# Delete a todo
# Print todos list with barcode of printed timestamp 
```
